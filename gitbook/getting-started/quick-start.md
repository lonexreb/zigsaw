# 🚀 Quick Start Guide

Get up and running with Zigsaw in under 10 minutes. This guide will walk you through creating your first automated workflow using our revolutionary chat interface.

## 📋 **Prerequisites**

- A Zigsaw account (sign up at [figsaw.dev](https://figsaw.dev))
- Access to services you want to automate (Gmail, GitHub, etc.)
- 5 minutes of your time!

{% hint style="info" %}
**New to automation?** Don't worry! Zigsaw is designed to be intuitive for everyone, regardless of technical background.
{% endhint %}

## 🎯 **Step 1: Access the AI Workflow Creator**

1. **Log in** to your Zigsaw dashboard
2. Click on the **"AI Creator"** tab (it's the default landing page)
3. You'll see a chat interface with example prompts

![AI Workflow Creator Interface](../.gitbook/assets/ai-creator-interface.png)

## 💬 **Step 2: Describe Your Workflow**

Instead of complex setup, just tell Zigsaw what you want to automate in plain English:

### Example Workflows to Try:

**📧 Email Automation:**
```
"When I receive an email with 'urgent' in the subject, send me a Slack notification"
```

**📝 Content Processing:**
```
"When someone uploads a document, summarize it with AI and email the summary to my team"
```

**🐙 GitHub Automation:**
```
"When a pull request is created, have AI review the code and post a summary as a comment"
```

**📅 Meeting Automation:**
```
"When a calendar event is created, generate an AI briefing and send it 1 hour before the meeting"
```

## 🤖 **Step 3: Let AI Generate Your Workflow**

1. **Type or select** one of the example prompts
2. **Click Send** or press Enter
3. **Watch** as Zigsaw's AI creates your workflow in real-time
4. **Review** the generated workflow structure

The AI will automatically:
- Choose the right node types
- Set up proper connections
- Configure basic settings
- Estimate execution time

## ❓ **Step 4: Answer Configuration Questions**

Zigsaw may ask clarifying questions to perfect your workflow:

- **API Keys**: "Which AI provider would you like to use?"
- **Permissions**: "Which email address should receive notifications?"
- **Settings**: "How often should this workflow run?"

Simply answer in the chat - no technical configuration needed!

## 👀 **Step 5: Preview Your Workflow**

1. **Click "View Workflow"** to see the visual representation
2. **Review** the workflow steps and connections
3. **Check** required permissions and API keys
4. **Verify** the estimated execution time

![Workflow Preview](../.gitbook/assets/workflow-preview.png)

## 🚀 **Step 6: Deploy and Execute**

1. **Click "Execute Workflow"** when you're ready
2. **Watch** real-time execution with live progress updates
3. **Monitor** results and any output data
4. **Save** your workflow for future use

{% hint style="success" %}
**Congratulations!** You've just created and deployed your first automated workflow using natural language. No coding required!
{% endhint %}

## 🔧 **Step 7: Fine-tune (Optional)**

Want to make adjustments? You have two options:

### Chat-based Refinement:
```
"Can you add a delay of 30 minutes before sending the email?"
"Change the AI model to Claude 4 for better analysis"
```

### Visual Editor:
- Switch to the **Visual Builder** tab
- Drag and drop to modify nodes
- Adjust settings through intuitive forms
